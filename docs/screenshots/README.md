## Usage

These screenshots are embedded in the main README.md to showcase the website's features and capabilities.

## Regenerating Screenshots

To regenerate these screenshots, run:

```bash
python scripts/generate_screenshots.py
```

This script uses Selenium with headless Chrome to capture screenshots of the static website.
